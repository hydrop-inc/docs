# Oktopus Docs

Documentación de [Oktopus](https://www.oktopus.lat) — el riel de pago contra entrega (COD) para LATAM, agent-native. Construida con [Mintlify](https://mintlify.com).

- Producción: docs.oktopus.lat
- Contenido: MCP server (47+ tools), OAuth 2.1, checkout COD (widget/headless/eject), API REST.

## Desarrollo local

```bash
npm i -g mint
mint dev
```
